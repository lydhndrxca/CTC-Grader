## 🚀 Quick Start

### Run the Web Server
```bash
cd D:\Projects\CTC_Grading\web
set OPENAI_API_KEY=sk-proj-your-api-key-here
node server.js
```

**Or** use the batch file:
```bash
cd web
START.bat
```

**Access:** http://localhost:3000
